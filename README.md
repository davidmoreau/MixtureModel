# MixtureModel
This series of scripts accompanies the paper "When Averaging Goes Wrong: The Case for Mixture Model Estimation in Psychological Science".

Please cite as:

Moreau, D., & Corballis, M. C. (2019). When averaging goes wrong: The case for mixture model estimation in psychological science. Journal of Experimental Psychology: General, 148(9), 1615-1627.​​ ​
